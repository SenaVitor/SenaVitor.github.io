Projeto Show do Pedrão

Esse projeto é um quiz com 10 perguntas sobre informática que teve como inspiração o programa "Show do Milhão". Foi o meu trabalho final da matéria de P.O.O JAVA. Tinha como intuito testar minhas habilidades com a linguagem em Programação Orientada a Objetos. Apesar do professor não ter exigido o uso de interface gráfica, busquei utilizá-la para aprimorar meus conhecimentos. O programa apresenta uma tela inicial com 2 botões, um para começar o jogo e outro para tocar a música do "Show do Milhão". Após clicar em "Começar" aparece um campo para o usuário inserir o seu nome, após isso dá-se início ao quiz, quando quando o usuário terminar de responder as 10 perguntas, ele é redirecionado para uma tela que mostra os erros e acertos do usuário.

Projeto Hotel Ruanda

Esse projeto é um Sistema de Hotel com as opções de cadastrar, atualizar, visualizar e excluir os dados dos clientes e dos funcionários do hotel. O programa foi feito para examinar os meus conhecimentos em relação a conexão entre a linguagem de programação JAVA e o Banco de Dados, para isso o professor nos pediu para fazer um programa que fizesse um CRUD básico. O programa inicia em uma tela de login, na qual o usuário deve inserir o login e a senha pré-definidos no Banco de Dados, se os dados inseridos não forem equivalentes aos do banco de dados, aparecerá uma janela informando que o login ou a senha estão incorretos. Após inserir os dados corretamente o usuário poderá escolher gerenciar os hóspedes ou funcionários. Irei mostrar apenas as telas dos hóspedes, já que foram as que eu programei. Após escolher a opção "Hospede" o usuário poderá escolher entre as opções de cadastrar, atualizar, visualizar e cancelar a reserva do hóspede.
