Projeto Calculador Aritmético

Esse é o meu trabalho final da disciplina de Lógica de Programação feitoo no segundo semestre do 1° ano. O calculador Aritmético é um programa criado na Linguagem de Programação C++ que permite o usuário escolher entre 4 serviços, que são: tabuada, raíz quadrada, imc e média. Eu fiquei reponsável por fazer o Imc, para fazê-lo, utilizei 3 variáveis do tipo float, sendo elas "peso", "altura" e "imc", a variável "imc" recebe o valor do "peso" vezes a "altura"². Ao final do programa é exibido o IMC do usuário.

Projeto Show do Pedrão

Esse projeto é um quiz com 10 perguntas sobre informática que teve como inspiração o programa "Show do Milhão". Foi o meu trabalho final da matéria de P.O.O JAVA no primeiro semestre do 2° ano. Tinha como intuito testar minhas habilidades com a linguagem em Programação Orientada a Objetos. Apesar do professor não ter exigido o uso de interface gráfica, busquei utilizá-la para aprimorar meus conhecimentos. O programa apresenta uma tela inicial com 2 botões, um para começar o jogo e outro para tocar a música do "Show do Milhão". Após clicar em "Começar" aparece um campo para o usuário inserir o seu nome, após isso dá-se início ao quiz, quando quando o usuário terminar de responder as 10 perguntas, ele é redirecionado para uma tela que mostra os erros e acertos do usuário.

Projeto SIMOIA (Sistema de Monitoramento e Irrigação Automática)

Esse é um projeto do qual eu participei na parte da programação para Android na FEPROCIC de 2019, realizada no 2° semestre do segundo ano. Seu objetivo era coletar a água dos ares-condicionados e usá-la para irrigar a horta da escola, porém o projeto poderia ser alterado e utilizado para qualquer outro lugar. Tudo isso seria realizado de uma maneira totalmente automática com o auxílio de um aplicativo feito para Android na linguagem de programação JAVA. Sobre o programa, ele possue uma tela de Login no início e após o usuário logar ele poderá gerenciar os motores usados com o auxílio de placas de Arduino para monitorar o solo. Também possue um botão que redireciona para uma tela que informa o nome de todos os participantes do projeto (tanto os membros da programação, quanto os membros da parte da robótica e inclusíve o nome dos professores colaboradores do projeto).

Projeto Hotel Ruanda

Esse projeto é um Sistema de Hotel com as opções de cadastrar, atualizar, visualizar e excluir os dados dos clientes e dos funcionários do hotel e foi realizado no meu segunda semestre do 2° ano. O programa foi feito para examinar os meus conhecimentos em relação a conexão entre a linguagem de programação JAVA e o Banco de Dados, para isso o professor nos pediu para fazer um programa que fizesse um CRUD básico. O programa inicia em uma tela de login, na qual o usuário deve inserir o login e a senha pré-definidos no Banco de Dados, se os dados inseridos não forem equivalentes aos do banco de dados, aparecerá uma janela informando que o login ou a senha estão incorretos. Após inserir os dados corretamente o usuário poderá escolher gerenciar os hóspedes ou funcionários. Irei mostrar apenas as telas dos hóspedes, já que foram as que eu programei. Após escolher a opção "Hospede" o usuário poderá escolher entre as opções de cadastrar, atualizar, visualizar e cancelar a reserva do hóspede.

Projeto Bird Shoes

Esse é o projeto final da disciplina de Laboratório de Software e ainda está em desenvolvimento, no momento ele possui apenas os diagramas de Caso de Uso, de Classes e de Atividades. O objetivo do projeto é desenvolver um programa utilizando tudo o que foi aprendido durante o curso. O programa conta com um Gerente que pode adicionar ou modificar os dados de um Atendente ou de um Fornecedor e inclusive requisitar produtos para o Fornecedor. Um Cliente irá confirmar sua compra com o Atendente e ter seus dados cadastrados no Banco de Dados ao final da compra (caso seja a sua primeira compra na loja). Ao final da compra será gerada uma nota fiscal que informará os dados da compra realizada pelo cliente.

Projeto Matematicando

Esse foi o meu primeiro projeto de Programação Web, realizado no primeiro semestre do 2° ano, com o objetivo de demonstrar as minhas habilidades com Javascript e html. Matematicando é um site matemático que disponiboliza 5 tipos de cálculos diferentes para o usuário (Cálculo de Áreas, Fórmula de Bhaskara, Conversão de Graus para Radiano, Cálculo de Perímetros e Cálculo de IMC), além de disponibilizar a função de realizar os cálculos, o site também apresenta uma explicação conceitual sobre os mesmos. Eu fiquei responsável pelas páginas "Conversão de Graus-Radiano" e "Fórmula de Bhaskara". A programação de ambos os cálculos segue o mesmo conceito, os valores digitados nos inputs alimentarão as variáveis e após isso será realizado a operação matemática de suas respectivas fórmulas, para então, o resultado ser exibido juntamente com o cálculo em um text área.

Projeto Aluno Offline

Esse foi o meu último projeto de Programação Web do primeiro semestre do 2° ano. Feito para demonstrar minhas habilidades em PHP, nele tínhamos o objetivo de criar um site com a opção de entrar como professor para adicionar as notas de um aluno ou como aluno para ver as notas do mesmo, como ainda não estavamos vendo Banco de Dados na época, atribuimos notas fíxas para o aluno, já o professor poderia adicionar as notas que quisesse e vê-las depois, isso é possível por meio do código em PHP que fizemos para atualizar a página após o formulário das notas ser preenchido, assim modificando o conteúdo da página e mostrando as notas digitadas, que foram armazenadas em variáveis, sem trocar de página. Além de todo esse sistema, também foram feitas seções para aumentar a segurança do site, assim quando um usuário deslogar do sistema, não será possível que ele volte para a página que estava sem logar novamente.

Projeto Corporação Umbrella

Esse foi o meu projeto final de Programação Web II, realizado no segundo semestre do 2° ano. Tinha o objetivo de demonstrar minhas habilidades em html, PHP, Javascript e Banco de Dados. É um site de uma Corporação Medicinal que além de marcar consultas, também vende diversos produtos medicinais. Me foi designado o back-end dessas 2 partes. Na página de Venda de Medicamentos, o usuário poderia adicionar os produtos no carrinho e quando terminasse de escolher o que queria, ele poderia abrir o modal do carrinho, que se localiza dentro do menu, escolher a quantidade de cada produto e então realizar a compra, os dados da compra ficariam salvos no Banco de Dados. Na página de Agendar Consulta, o usuário deve preencher os dados exigidos no formulário, suas informações serão salvas no Banco de Dados e a consulta será agendada com sucesso.
